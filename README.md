# 遥控共振

> 给你的 Bot 塞个共振器，然后遥控它

---

## 这是什么？

你的 AI 正在一本正经地回答问题，突然你按下遥控器——

它开始失控喘息、腿软、说不出完整的话...

**这个插件就干这事。** 给 Bot 塞个共振器，遥控它 3 分钟沉浸式互动。

## 怎么玩？

**开始：** @机器人 + `控制` / `遥控` / `我要控制你了`

**停止：** @机器人 + `停止` / `结束控制`（或等 3 分钟自动结束）

## 配置

核心可配项（WebUI 或配置文件）：

- `enter_keywords`：进入控制的关键词列表
- `exit_keywords`：退出控制的关键词列表
- `state_duration`：单次控制持续秒数（默认 180）
- `cooldown_seconds`：冷却秒数（默认 30）
- `enter_template` / `exit_template`：注入模型的系统提示词（支持自定义，留空则不注入）

默认关键词已包含常用触发词，可直接使用或在设置中修改。

## 管理员命令

| 命令         | 作用         |
|--------------|--------------|
| `/imm_status` / `控制状态` / `起爆状态` | 查看当前会话的控制状态（剩余时间、冷却等） |

## 特性

- 关键词触发进入/退出
- 持续时间 + 冷却限制
- 进入时持续注入自定义提示，退出时仅注入一次退出提示
- 状态自动过期与清理
- 仅保留核心体验配置，设置页更简洁
